Thanks for installing the Angel Umbrella (Mntone WinUI 3 Library).

Don't forget to set `Generic.xaml` as your Application resources in App.xaml:

<Application>
	<Application.Resources>
		<!-- WinUI 3 -->
		<XamlControlsResources xmlns="using:Microsoft.UI.Xaml.Controls" />
		<ResourceDictionary Source="/Mntone.AngelUmbrella/Themes/Generic.xaml" />
	</Application.Resources>
</Application>

See https://github.com/mntone/AngelUmbrella for more information.
